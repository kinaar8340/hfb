# HFB Glossary

| Term | Definition |
|------|------------|
| **HFB** | Hopf Flux Bubble — emergent effective-metric playground from defects + flows + Hopf topology |
| **Analog gravity** | Propagation of quasiparticles/waves in an effective curved spacetime set by a background medium or flow |
| **Acoustic metric** | Line element for sound in a fluid with velocity **v** and sound speed cₛ: ds² = −(cₛ²−v²)dt² − 2**v**·d**x** dt + d**x**² |
| **Ergoregion** | Region where co-rotating modes are dragged faster than wave speed — analog of GR ergosphere |
| **Topological defect** | Stable field singularity from symmetry breaking; carries quantized winding number k |
| **Conformal factor Ω** | Log-scale factor in g = e^{2Ω} δ; defect density λ sources ΔΩ = −λ |
| **Cosmic string analog** | Nematic or optical vortex line deflecting rays with angular deficit Δφ ∝ Δn/n |
| **Hopf fibration** | S³ → S² fibration; linked circle preimages — template for toroidal flux walls |
| **Hopfion** | Topological soliton with nonzero Hopf invariant (linked field preimages) |
| **Flux bubble** | Toroidal wall of defects/flow/index gradients modeling contraction ahead, expansion behind |
| **Shift profile** | Effective propagation bias (Alcubierre-*like* analog, not literal GR) |
| **Nematicon** | Self-guided optical soliton in a nematic liquid crystal |
| **LG mode** | Laguerre-Gaussian beam with orbital angular momentum ℓ |
| **BEC analog** | Bose-Einstein condensate used as tabletop acoustic/optical analog medium |
| **Electro-vibrational control** | Boundary drive coupling electrostatic dual shells to vibrational modes of the flux medium |
| **Dual charge envelopes** | Concentric shells with opposing surface charge ±σ; gap forms a capacitive control layer |
| **Phase-alignment order ψ** | Lock quality between charged-vibration drive and medium resonance (0–1); supercritical ψ nucleates voids |
| **Hemi-void / gourd bubble** | Asymmetric void wall elongated orthogonally; rear hemi stores, front is tapered for directional bias |
| **Flux flywheel** | Hybrid rear-hemi reservoir: electrostatic storage + flux/twist tension (rubber-band packing) |
| **Slingshot release** | Controlled detuning that dumps flywheel energy into a forward shift boost (directional metric reconfiguration) |
| **Observer synchronization** | Resonant entrainment of an internal clock / operator with the macro medium “hum” as control feedback |
| **Charged vibration** | Time-harmonic shell displacement modulated by capacitive |E|; couples to Hopfion / defect density |
| **Flux transducer** | Craft-local device between internal systems and rear-hemi void: quantifies stored energy, reverts flux channels, meters release intensity |
| **Energy ledger** | Explicit three-channel bookkeeping: electrostatic + twist + geometric tension (not global curvature flux) |
| **Channel reversion** | Polarity flip of flux pathways from inbound store (+1) to outbound forward dump (−1) on release command |
| **Flow intensity / throttle** | 0–1 control of how much and how fast ledger energy is dumped as directed impulse |
| **Directed impulse** | Metered forward boost after reversion × intensity × release efficiency — the slingshot kick |
| **Pre-charge** | Active motor path: transducer drives dual envelopes to bank electrostatic energy on command |
| **Pre-twist** | Active motor path: parametric inject of azimuthal flux into rear-hemi flywheel resonators |
| **Pump intensity** | 0–1 throttle for pre-charge / pre-twist drive (independent of release intensity) |
| **Ready state** | Rear hemi at target ledger energy; channels hold/top-up until release command |
| **Σ pumped** | Cumulative actively injected energy (motor path) — fidelity metric vs passive collection |
| **Σ passive** | Cumulative ambient-collected energy (generator path) |
| **Pumped efficiency** | Motor share of intake: Σ pumped / (Σ pumped + Σ passive) |
| **READY hysteresis** | Band below target_energy before READY drops — prevents pump chatter |
| **Storage breakdown** | Diagnostic: channel %, total_stored, capacity fraction, motor vs passive |
| **Craft dynamics** | Payload / hull equations of motion driven by transducer impulse + ledger (not global GR) |
| **Directed Δv** | Axial velocity kick: coupling · J / m, with optional ES-channel boost at dump |
| **Store recoil** | Small rearward reaction while actively loading the rear hemi |
| **Mission** | Coupled engine cycle + craft integration (`simulate_mission`) |
